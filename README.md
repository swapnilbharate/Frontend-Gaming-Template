# 🎮 Frontend Gaming Template

A modern and responsive Gaming Website Template built using React.js and Vite. This project features a visually appealing gaming-themed UI with sections for games, tournaments, reviews, and user engagement.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/swapnilbharate/Frontend-Gaming-Template.git
```

Navigate to the project folder:

```bash
cd Frontend-Gaming-Template
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

## 📦 Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
